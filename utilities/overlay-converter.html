1. Drag & drop or select image files (SVG, PNG, JPG)
2. Click "Convert All to Base64" to process the images
3. Each image will download as a separate JSON file
4. Place the JSON files in assets/overlays/ folder
5. Reload the app to see them in the overlay dropdown
